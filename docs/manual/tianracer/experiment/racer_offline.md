# 比赛二 F1TENTH 线下竞速赛

## 【比赛目标】

- 参照 2021ROS 暑期学校 F1TENTH 线上仿真赛，与比赛一的形式，可自行组织学生完成线下比赛，学生自行进行调参，以跑 3 圈所用时间为竞技标准
- 比赛形式为线下
- 老师可组织学生自行搭建赛道，自定比赛规则

## 【注意事项】

线下赛与线上赛虽然形式类似，但实际操作起来还是会遇到一些问题，这就考验到学生对整车、结构、电路、算法、部署等的调节能力，同时要考虑赛道的搭建等。
