# Tianracer F1TENTH 仿真实例 

### 1.简介

Tianracer_gazebo 是一个基于 Gazebo 的 Tianracer 的仿真环境，在这个仿真环境中，仍然可以通过阅读 Tianracer 使用手册，来进行仿真调试。  

### 2.如何运行 

使用下面的命令即可运行仿真环境

```shell
roslaunch tianracer_gazebo demo_tianracer_teb_nav.launch
```
