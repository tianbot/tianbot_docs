# PROBOT 仿真实例

## **PROBOT 仿真演示**
本演示仿真示例的内容是六轴机械臂的执行末端安装有一个自由度的夹爪，由程序操作整个机械臂让末端假抓移动桌子上的可乐罐，完整的例程和代码，对 ROS 机械臂应用学习具有很实用的借鉴价值。
* 打开终端（ctrl + alt + t），输入以下命令，启动 PROBOT 例程
```
roslaunch probot_grasping probot_anno_grasping_demo.launch
```
