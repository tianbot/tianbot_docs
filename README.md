# 如何运行本项目

1. 安装node.js

- [Node.js 开发环境安装之 Linux](https://zhuanlan.zhihu.com/p/144885938)
- [node-v18.17.0-linux-x64 下载链接](https://cdn.npmmirror.com/binaries/node/v18.17.0/node-v18.17.0-linux-x64.tar.xz)

2. 安装pnpm

```shell
npm install -g pnpm
```

3. 安装node_modeules

```shell
pnpm install
```

4. 本地运行预览
```shell
pnpm run docs:dev
```
# Contact

5. 编译语法检查
```shell
pnpm run docs:build
```
